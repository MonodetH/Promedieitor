Variable CONDICIONAL

Los parámetros que recibe tiene son de la forma:

X , condicion [& [ condicion]] : valor [, ...][; valor_por_defecto]

condicion: { { >|>=|<|<=|==|=|!= }(espacio)Y }
donde X e Y pueden ser Variables, Evaluaciones, enteros o decimales, siendo X la variable principal a la cual comparar e Y una variable con la cual X es comparada.
Si el valor por defecto no es especificado, se asume 0
a continuación algunos ejemplos:
NS,<= 55:0;NS (reprobación absoluta)
NS,>= 44 & < 55: NFG; NS (nota con global)
X, < Y: X, > Y: Y, == Y: X ( mínimo de X e Y )
